Planificador Estratégico AI

Python | Streamlit | Gemini 2.0 Vision

Arquitectura del Flujo

El sistema recibe un archivo (PDF/JPG) arquitectónico que se codifica y envía vía API a Gemini Vision mediante un Prompt Estructurado. El LLM devuelve un formato JSON validado con coordenadas sugeridas que la capa frontal (Streamlit) interpreta para renderizar dinámicamente sobre un mapa (Canvas).

Complejidad Técnica Resuelta

  • Manejo avanzado de session_state para persistir las coordenadas de los equipos insertados por el usuario sin recargar el DOM.
  • Algoritmo matemático de escalamiento interactivo: Convierte de forma precisa "píxeles de pantalla" a "metros reales" según la calibración manual del usuario.
  • Cálculo volumétrico dinámico (Radio = Área / Altura / Capacidad del equipo) ajustando el círculo de cobertura en tiempo real.

Impacto en CX / Negocio

Digitalizó y profesionalizó un proceso de preventa manual. Los asesores ahora generan propuestas técnicas exportables a PDF corporativo en minutos, aumentando la confianza técnica del cliente B2B.

Bot Omnicanal & Kanban CX

Webhooks | Meta Cloud API | n8n | Airtable

Arquitectura del Flujo

Los mensajes ingresan vía Webhooks desde Meta API hacia un orquestador lógico (n8n / Flask). El motor evalúa palabras clave y lógica condicional para segmentar la atención (UENs). La data se persiste de forma bidireccional mediante APIs REST hacia una base relacional (Airtable), que alimenta un tablero Kanban Frontend.

Complejidad Técnica Resuelta

  • Gestión de payloads asíncronos para asegurar que el bot no bloquee la recepción masiva de mensajes y asigne prioridades.
  • Diseño de enrutamiento condicional que identifica si el cliente pertenece a distintas ramas del negocio.
  • Implementación de la gestión de estados (Nuevo, En Proceso, Resuelto) con disparo automatizado de encuestas (CSAT) basado en cambios de registro en la DB.

Impacto en CX / Negocio

Redujo drásticamente los tiempos de primera respuesta (SLA). El tablero visual empoderó a los agentes operativos, eliminando el "caos" de una bandeja de WhatsApp estándar.

Sincro IA (App Store)

SwiftUI | EventKit | SwiftData | Gemini API

Arquitectura del Flujo

Aplicación nativa iOS bajo el patrón de diseño MVVM. El modelo interactúa con el framework EventKit de Apple para leer la agenda del usuario. La vista recolecta un "volcado mental" que se envía a Gemini de manera asíncrona, y la IA retorna una matriz de tareas que el controlador inyecta de vuelta en el calendario de forma segura.

Complejidad Técnica Resuelta

  • Lógica de resolución de conflictos temporales: El algoritmo evalúa los bloques de EventKit para asegurar matemáticamente que la IA jamás sobrescriba o empalme reuniones de trabajo existentes.
  • Implementación segura de concurrencia (async/await) para evitar que la interfaz se congele durante el llamado a la API.
  • Diseño UX centrado en neurodivergencia: Mitigación de sobrecarga cognitiva estructurando la UI para no abrumar al usuario.

Impacto en CX / Negocio

Producto publicado comercialmente en la App Store, superando las estrictas validaciones de rendimiento y privacidad de Apple.

CRM & Dashboard Financiero

React | Hooks | CSS Grid | Data Analytics

Arquitectura del Flujo

SPA (Single Page Application) sin backend pesado (Client-Side Rendering). Un módulo de carga recibe archivos .csv en bruto, normalizando miles de filas en memoria. El estado global distribuye esta data a múltiples vistas (Pipeline, Cortes, Histórico) reaccionando en tiempo real a los filtros del usuario.

Complejidad Técnica Resuelta

  • Optimización extrema de renderizado utilizando useMemo y useCallback para evitar bloqueos del navegador al recalcular matrices de datos grandes (MRR, ARR, MoM).
  • Algoritmo de agrupación multidimensional: Transforma registros planos en objetos anidados indexados por Cliente, KAM, Trimestre y Unidad de Negocio.
  • Motor de lógica de negocio que calcula dinámicamente bonos comisionables basados en reglas condicionales variables.

Impacto en CX / Negocio

Otorga visibilidad financiera instantánea a la gerencia. Sustituyó el uso de múltiples hojas de cálculo pesadas por una interfaz analítica única y centralizada.
